## Notes d'utilisation pour l'IA

1. **Toujours utiliser la colonne "Prix retenu"** (fourchette haute)
2. Ces prix incluent fourniture + pose sauf mention contraire
3. Ajouter **30% de marge** au prix retenu pour obtenir le prix de vente
4. En Île-de-France, appliquer un coefficient **×1.15 à 1.20** sur les prix
5. Si un ouvrage n'est pas dans cette table, utiliser web search pour trouver un prix de référence et toujours prendre la fourchette haute
6. Ces prix sont indicatifs — les anciens DPGF du client (quand disponibles) seront la meilleure source
